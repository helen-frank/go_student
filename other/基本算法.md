## 1.1 排序

### 1.1.1 快排 - 分治

1. 确定分界点：x点   `q[(l+r+1)/2]`
2. 调整区间：x左边都小于x , x右边都大于x
3. 递归处理左右两段

处理办法：额外开辟两个数组，`双指针法`

模板

```go
func QuickSort(q []int, l, r int) {
	if l >= r {
		return
	}
	x := q[(l+r)/2]
	i := l - 1
	j := r + 1
	for i < j {
		i++
		j--

		for q[i] < x {
			i++
		}
		for q[j] > x {
			j--
		}
		if i < j {
			q[i], q[j] = q[j], q[i]
		}
	}
	QuickSort(q, l, j)
	QuickSort(q, j+1, r)
}
```
